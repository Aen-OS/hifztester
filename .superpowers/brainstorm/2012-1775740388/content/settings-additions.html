Design notes:

  • Game Length — toggle between question count and time limit. "Unlimited" checkbox preserves current behavior. Defaults to 20 questions.
  • Focus Mode — replaces the manual scope selector when toggled on. One-tap "Practice Weakest" auto-selects, or user picks granularity (juz/surah) and overrides from the ranked list.
  • Confidence colors — red <50%, amber 50-75%, green >75%
  • New user state — Focus Mode shows "not enough data" until they've played enough sessions. Minimum: at least 1 session with data.
  • The "Manual" tab shows the existing scope selector unchanged.
